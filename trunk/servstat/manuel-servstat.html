0 - Préambule

Bonjour et merci de prendre la peine de lire ces quelques infos pour que vous puissiez au mieux profiter de notre petit module ... :)

Ce module sert à monitorer un ou plusieurs serveurs de jeux à partir d'un site Web de type NPDS

Fonctionnalités de base du module :

  1. Bloc :
    • affichage de la map en cours sur chaque serveur
    • affichage des principales caractéristiques du serveur
    • affichage de la liste de joueurs connectés
  2. Page principale :
    • affichage des serveurs favoris (caractéristiques et pages de stats)
    • interrogation de n'importe quel serveur
    • liste des joueurs actuellement en jeu

Fonctionnalités du menu administration :

  • gestion des favoris
    • Ajouter/modifier/supprimer un serveur de la liste des favoris
    • Enregistrer le lien vers la page web de statistique des serveurs
  • Configurer l'affichage du bloc en première page

1 - Installation

  1. Dézipper l'archive à la racine du site web
  2. Editer le fichier serv_config.php pour rajouter le chemin relatif de servstat par rapport au nom de domaine
  3. Exécuter le script servstat.sql
  4. Créer un bloc via la page d'administration de NPDS:
    • choisir un coté (bloc droit ou bloc gauche)
    • choisir un titre pour le bloc :
       Exemple : Servstat 2.0
    • Placer le code suivant dans le contenu du bloc:
       include#modules/servstat/servstat.php
  5. Via le menu d'administration ("Editer les fichiers de configuration"), éditer le fichier NPDS "extend-modules.txt" et lui ajouter les lignes suivantes :
     [module]
      [nom]Administration Servstat[/nom]
      [ModPath]servstat[/ModPath]
      [ModStart]admin/admin_serv[/ModStart]
      [niveau]radminsuper[/niveau]
     [/module]
    Puis sauvegarder
Votre module est maintenant installé, vous pouvez passer à son administration

2 - Administration

/!\ ATTENTION: seul un administrateur du site de niveau "super admin" peut configurer le module via la console d'administration /!\

  1. Nom Affiché:
    C'est le nom qui apparaîtra en première page dans le bloc au dessus de l'image du serveur. Ce nom peut être différent du nom réel du serveur.
      Ex: nom réel du serveur : [FR]::Fun.Xp Plan Of Attack =|20|= - wWw.FunXp.net -
            nom affiché : Plan Of Attack =|20|=
  2. Adresse : Port :
    Entrez ici l'adresse du serveur. Celle ci peut être du type IP ou URL. Le port en revanche est obligatoirement de type numérique
      Ex : Adresse : GamersLobby.net
            port : 20300
      Ex : Adresse : 213.251.144.36
            port : 27015
  3. Jeu :
    Sélectionner le jeu correspondant au serveur
  4. URL des statistiques :
    Si vous disposez d'un site web où l'on peut consulter les statistiques de jeu de votre serveur, entrez la ici. Cette possibilité intéresse entre autres les joueurs sur Half-Life 1 et 2 et ses mods (Counter-Strike, Day Of Defeat, etc.)
  5. Etat/Ordre d'affichage :
    Vous pouvez ici configurer l'ordre d'affichage des serveurs dans le bloc servstat Ils s'afficheront dans l'ordre croissant sélectionné. Si vous sélectionnez "caché" pour un serveur, il n'apparaîtra pas dans le bloc.
  6. Image map :
    Affiche ou cache dans le bloc l'image de la map jouée sur le serveur. /!\Attention/!\ : si vous cachez l'image , vous n'aurez plus accès à la petite fenêtre s'affichant par dessus et qui contient les infos principales du serveur
  7. Nom map :
    Affiche ou cache la ligne indiquant le nom de la map en cours sur le serveur ainsi que le nombre de joueurs actuellement présents.
  8. Liste joueurs :
    Affiche ou cache la liste déroulante contenant le nom de tous les joueurs actuellement connectés au serveur. S'il n'y a aucun joueur connecté, cette liste n'apparaîtra pas
  9. Commentaire :
    Vous pouvez ici rentrer un commentaire au sujet de votre serveur. Ce commentaire ne sera disponible que pour l'administrateur.

3 - Accès au module

Ici rien n'est à configurer. Les données suivantes sont à titre indicatif:

  • Accès au module servstat se fait via une url de type :
    modules.php?ModPath=servstat&ModStart=serv_detail&ip=213.186.50.2&port=27015&game=Counterstrike
    Il peut être pratique d'ajouter au menu principal de NPDS un lien de ce type afin d'accéder directement à la page principale du module
  • Accès direct à la partie administration se fait via une URL du type :
    admin.php?op=Extend-Admin-SubModule&ModPath=servstat&ModStart=admin/admin_serv
    Il peut être pratique d'ajouter au menu d'administration de NPDS un lien de ce type afin d'accéder directement à la partie adminstration du module
